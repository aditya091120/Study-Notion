# StudyNotion 🎓

StudyNotion is a **full‑stack MERN-based EdTech platform** that allows users to **create, consume, and rate online courses**. It includes **secure authentication, role-based access, course management, payments, and dashboards** for students and instructors.

---

## 🚀 Live Demo

* **Frontend:** [https://study-notion-mern-frontend.vercel.app/](https://study-notion-mern-frontend.vercel.app/)
* **Backend:** [https://studynotion-mern-0763.onrender.com](https://studynotion-mern-0763.onrender.com)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* JavaScript (ES6+)
* Redux Toolkit
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Bcrypt
* Cloudinary (Media Storage)

### Payments & Services

* Razorpay (Course Payments)
* Nodemailer (Email & OTP Verification)

### Deployment

* Frontend: **Vercel**
* Backend: **Render**
* Database: **MongoDB Atlas**

---

## ✨ Key Features

### 🔐 Authentication & Authorization

* User Signup & Login
* Email Verification using OTP
* Forgot / Reset Password
* JWT-based authentication
* Role-based access control (Student / Instructor / Admin)

---

### 👨‍🎓 Student Features

* Browse all available courses
* View course details
* Purchase courses using Razorpay
* Enroll in paid courses
* Watch course videos
* Track course progress
* Add & manage courses in cart
* Rate & review courses
* Student dashboard

---

### 👨‍🏫 Instructor Features

* Instructor dashboard
* Create new courses
* Add sections & subsections (lectures)
* Upload videos & thumbnails
* Edit / delete courses
* Publish or unpublish courses
* View enrolled students
* Track course performance

---

### 🛒 Payment Integration

* Secure Razorpay payment gateway
* Course purchase verification
* Payment success & failure handling
* Backend payment signature verification

---

### 📩 Email Services

* OTP email verification during signup
* Password reset email
* Transaction & account notifications

---

### 📁 Media Management

* Course thumbnails & videos stored on Cloudinary
* Secure video uploads

---

## 🔮 Future Enhancements
